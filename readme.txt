A responsive and intuitive plugin for using your ListShine signup forms
on your wordpress website!

== Description ==

We are proud to announce our latest plugin which enables you to embed all of the signup forms
you have created on your ListShine account. This can be done with a couple of clicks!

== Installation ==
It takes 3 steps!

1. Upload the folder containing the plugin files to the 'wp-content/plugins/'
    or you can install it through the WordPress plugin screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Start embedding!

== Usage ==

There are two ways you can embed your forms:

0. First time activating
    -If you have activated it for the first time, you will have to enter your
     ListShine API key in order to gain access to all the forms you've made in ListShine.
     You can acquire your unique API from the ListShine app under
     the Integration -> Wordpress tab.

1. Through shortcode
    - Click on ListShine signup forms on the admin menu
    - Locate the form you want and click on the Get shortcode button
    - After you obtain the shortcode just paste it in the page you want and
      the form will appear on the page

2. Through the ListShine button in the page editor
    - Start editing the page or post you want to embed your form in.
    - Find the purple LS button on the toolbar of the text editor
    - Choose the name of the form you want to embed
    - The shortcode will be inserted in the page and will appear the next time you start it
